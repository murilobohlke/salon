# Como Utilizar?

~~~
- Clonar o repositório:
$ git clone https://github.com/murilobohlke/salon.git

- Entrar no diretório:
$ cd salon/salon

- Instalar as dependências:
$ flutter pub get

- Iniciar o app: 
$ flutter run
~~~
